[新竹] 伊尹小築主廚私房料理

Tue Apr 30, 2019 by Super Cat in  Gourmet  Hsinchu, Chinese Food

餐廳名稱: 伊尹小築主廚私房料理 地 址: 新竹市東區綠水里園後街4號 營業時間: 9:00-17:00 17:30-21:00 週二公休 伊尹小築主廚私房料理 很多年前開在金山街，那時叫 阿拉牧原住民餐廳， 除了常見的熱炒弄得很好吃以外，還有原住民料理如炸蟋蟀，炸蜂蛹等， 生意還不錯，但不知怎麼了，改作素菜，然後又變回原來的熱炒， 生意也還是不錯，然後很奇怪，有天就收攤了，原地點變成美術教室的樣子。 知道伊尹小築在東園路這邊復出江湖 已經一陣子了， 今天就再來探一下，看手藝有無退步。 到了之後，才知道，原來可以把車停在東園國小，他們有特約。 不過我老婆已經找到一個絕佳的合法停車點，非常近，超厲害的， 幫伊尹小築省下停車費了。 裡面有很多原住民的擺設，挺有特色的， 用竹筒做成的茶壺，很能代表山上的生活， 伊尹小築 的菜單，其實蠻多常見的台式熱炒， 特色菜只有4道，但我今天沒勇氣吃蟋蟀跟蜂蛹， 阿拉牧拌菜，靈感應該是來自松柏長青，調味不錯， 照理說松柏長青是用大白菜，不過他改用高麗菜， 加入豬耳朵等，拌在一起，蠻好吃的。 西檸雞球，靈感應該是來自橙汁排骨，橙汁雞球等，這種新的調味風格也不錯。 金沙排骨，這是道蠻流行的菜，許多餐館都有，不過我覺得鹹蛋裹太少了，稍微遜色一點。 蟹黃三鮮，其實真正的蟹黃很貴，全台灣的餐廳幾乎都是用紅蘿蔔泥跟鹹蛋黃去模擬了， 我只有七、八年前在台北南京東路吃到是真正蟹黃做的，價格偏貴但值得，不過幾年後 經過，那餐廳也不見了，所以想吃真正的蟹黃豆腐煲是非常難的。 伊尹小築的蟹黃三鮮，當然也是沒蟹黃的那種，只是老闆啊，不要叫它蟹黃三鮮吧， 一些餐廳都改口海鮮豆腐煲了，您也盡快改名吧。 Anyway, 伊尹小築的蟹黃三鮮 算是好吃。 選了不常見的酒釀蝦球，蝦仁很大顆，跟蟹黃三鮮的蝦仁不同大小，主廚有講究，讚! 味道不錯。 最後，店家送個甜點。 離開伊尹小築的時候，發現一隻貓，我女兒堅持要拍照，說它很可愛， 各位看這一團黑可不可愛吧! [竹北] The BBQ House 竹北店

Thu Apr 25, 2019 by Super Cat in  Gourmet  Hsinchu county, Zhubei, Steak

這間BBQ House，是在新竹市站穩後，開了竹北店， 食物偏貴，但食材用的不錯，算是好吃，偶而吃一次也還不錯。 裝潢還算新! 牆壁中間有個大大的高亮度招牌， 第一次來，簡單試一下，小菜選炸牛肚，湯選了蘑菇卡布奇諾湯， 湯好喝，炸牛肚普通。 松露大寬麵 這個麵好吃。 牛排是這間店的特色，有個專門放肉的玻璃冷凍櫃， 先挑肉的種類，然後選擇重量，想吃幾克就幾克， 今天挑 紐約客牛排，其實我覺得菲力牛排太瘦， 紐約客或是莎朗這種帶點油花，帶點筋的比較好吃。 主廚蠻會煎的，直接吃就不錯了，沾他附的牛排醬汁也不錯， 我試著用桌上的玫瑰鹽，感覺並沒有比較好吃。 Read more →

VirtualBox 6.x Shared Folder

Sun Apr 21, 2019 by Super Cat in  Computer  Virtual Box, Linux

前一版的 Ubuntu 16.04 LTS，它當 VirtualBox Guest OS 的時候， 可以安裝一個現成 virtualbox-dkms 套件，裝好之後， shared folder 就自動生效了，非常方便。 Ubuntu 18.04 LTS 問世之後，它的 virtualbox-dkms 套件 是針對 VirtualBox 5.2.18 及之後版本，當時 VirtualBox 也 是 5.2x 左右，一切也沒問題。 突然間，2018年12月 VirtualBox 6.0 降臨了，我直接將 VirtualBox 5.x 升級到 VirtualBox 6.0 ，跑一樣的 guest image， Shared Folder 就失效了。 查了很久，才發現是 virtualbox-dkms 套件只支援 VirtualBox 5.x 的關係，等了一陣子，看起來 Ubuntu 18.04 是沒打算升級 dkms 套件了， 那只好回到老方法，讓 VirtualBox Host 這邊插入虛擬的 Guest Additions 光碟片。 在 VirtualBox Host 這邊的 Menu Bar ，有一個 “裝置” 選項，裡面 有一個 “插入 Guest Additions CD 映像”。 Read more →
