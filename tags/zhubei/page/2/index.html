餐廳名稱: Yama Sweets 地 址: 新竹縣竹北市十興路一段231號 營業時間: 週四~週二 07:00~19:00 週三公休 電 話: 03 668 5090 因為團購券，知道了這家店 - Yama Sweets， 算是Coffee Shop 跟烘焙房的合體。因為 7:00 就開始營業，所以可以當早餐吃。 地點算是竹北莊敬北路靠近新埔的那一頭，稍微偏僻，可是也就好停車。 窗外目前還是綠地，景色不錯。 櫃台，有陳列一些精緻的蛋糕。 房子後方就是烘焙房了，前方則是咖啡店的部分。 不曉得是不是太早了，只有老闆娘一位忙來忙去，食物部分鹹食非常少，只有菜單裡面的輕食是鹹的， 其他都是甜甜的蛋糕、麵包。 今天飲料選了 熱紅茶，榛果拿鐵，抹茶歐雷。味道很正常。除了紅茶不甜的情況下不好喝，跟老闆娘要了糖包才改善。 燻雞手卷，味道還不錯。 叉燒可頌，這裡的叉燒不是廣式叉燒，比較像火腿，味道也蠻正常的。 牛肉起士捲餅，味道還不錯。 想說它是烘焙房，在玻璃櫃那邊選了一個 Yama千層，但吃起來略不習慣它的奶油，也許選別的會更好。 帳單如下，假日早餐想要悠閒悠閒地吃些好一點的，好像就是這樣的代價。 這家店 鹹食還算優秀，飲料普通，烘焙房的部分應該有一些好吃的甜食， 各位讀者就參考看看吧。 Read more →

[竹北] 竹北夜市

Sun Dec 01, 2019 by Super Cat in  Gourmet  Hsinchu county, Zhubei, Chinese Food

餐廳名稱: 竹北夜市 地 址: 新竹縣竹北市 環北路五段 與 新光二街 交叉口 營業時間: 週六~週日 17:30~23:30 電 話: 0960 071 638 這個週末因為一點意外，連續兩天晚上去竹北夜市。兩三年前，曾經收集過網友推薦過的竹北夜市攤販， 這週六來，竟然幾乎都沒有看到。週日來，一部分攤位跟週六擺在不一樣的地方，然後週日來比較多攤， 有多找到幾家 推薦名單中的攤位。 接下來，就介紹這兩天吃過的攤位，當然一定還有其他的優秀攤位，只是我剛好沒吃到。 這攤 珍菇棧脆皮杏鮑菇 很好吃，我們選傳統胡椒粉調味。兩天都有買。這攤兩天擺不一樣地方，害我以為它只來星期六，還好有找到。 湘記高粱酒湯汁煙錢 賣的是炭烤大腸包小腸，人很多，食物製作的過程很花時間。東西不錯吃，鹹度不錯，也帶有炭香。比 東元醫院附近那攤大腸包小腸好吃多了。 買了這攤的西瓜汁，攤位規模大，製作過程很熟練。雖然有看到她加糖、加冰塊去打果汁，最後裝杯時再加兩個冰塊，但是看的出來原料不錯，加一點糖也比較好喝。 紅葉滷味 賣的是冷滷。試買一些帶回家吃，其實還可以啦，但跟竹北名店的小政滷味、可口滷味 比起來，還是輸一點點。 這家 尚好現榨蔬果原汁 一樣是買個西瓜汁。沒認真看，似乎是沒加水、沒加糖。當然，果汁的味道就淡一點。 後山林記炸蛋蔥油餅 是 推薦名單當中的店家。 這家沒吃到，它星期六沒來，星期日要離開夜市了才來，照個像紀念一下。 德記香酥雞 也是推薦名單當中的店家，只有星期天有來擺攤。 老闆是喑啞人士，我口頭點餐後，老闆用手指著調味料牌子讓我選， 我選最傳統的胡椒口味。 雞肉炸的不錯，而且無骨，胡椒粉的味道如果是鹹一點的胡椒鹽，應該會再好吃一些。 這攤推薦。 凰巢鹹水雞，注意，是凰不是鳳喔，中間是個皇字。人時多時少，生意算不錯，也有網友推薦過。不過我覺得還是比不上竹北三民路那兩家鹹水雞名店。 西班牙油條，有網友推薦過，只有星期天有來擺攤。有位老外駐店，更增添西班牙的真實性。 這張照片不是要介紹攤販，是介紹手上這根西班牙油條，甜甜軟軟的，不錯吃耶。 炭烤珍珠玉米，覺得味道中規中矩，價格合理，有用旋轉的機器輔助烤玉米，可推薦。 夜市這種地方，最好還是能夠先查一下口碑，比較不會踩到雷，下次有機會，試試其他攤位，再來報告。 如果網友要推薦攤位，也很歡迎。 Read more →

[竹北] 沐樂早午餐

Sat Nov 30, 2019 by Super Cat in  Gourmet  Hsinchu county, Zhubei, Brunch

Updated on 2022/Feb/7: 已停止營業。 餐廳名稱: 沐樂早午餐 地 址: 新竹縣竹北市環北路五段410號 營業時間: 週一~週日 08:00~15:00 電 話: 03 555 6061 假日例行公事，睡到自然醒，再找個早午餐來吃，今天女王大人翻到一間剛開幾個月的早午餐 - 沐樂早午餐。 沐樂早午餐 在竹北夜市外圍這一帶，早上這裡稍微荒涼，停車還算OK 。這裡是個新蓋的大樓的一樓。 正門口寫著營業時間，似乎沒看到公休。 到的時間也許正好是它的用餐顛峰，一樓坐滿了，所以安排我們到二樓。每張桌子都擺著一個小的假綠色植物盆栽，上面寫著桌號。 從二樓往下照，拍這張照片的時間點已經是用餐快結束了，所以有空位了。 二樓的一個角落，餐具、醬料是自助的。 菜單，稍微奇怪的是 沒有咖啡。感覺跟一般咖啡店比起來，餐就比較多種，每間店的重點不一樣吧。 今天選的偏早午餐，下次再來試義大利麵或燉飯等。 總共點三個餐，最先上來的是飲料，我們都沒升級，都是預設的紅茶、牛奶。 沐樂經典牛肉堡。附餐選薯餅。女王選的。 沐樂培根牛肉堡。附餐選薯條。其實就是沐樂經典牛肉堡再加兩條培根，被女王搶走一條。 牛肉堡看那厚度及不規則形狀，應該是店家手工製作，吃起來也感覺也像。再把洋蔥、番茄、青菜一起夾起來，那個高度有點煩，只好盡量壓扁，努力張開大口咬下去。 味道蠻好吃的。 配角的薯條調味也是正常鹹香。 鮭魚歐姆蛋捲早餐。小朋友點的，我只有吃一口炒蛋，覺得也是不錯，嫩嫩鹹鹹的。 吃完覺得這樣的好吃度，配上那個價格，非常棒，跟竹北斑馬比起來，好吃又便宜。竹北斑馬很容易就要排隊，旁邊停車也困難，以後真的是要改成 沐樂早午餐 吃早午餐了。 很棒的早午餐店，推薦。 Read more →
