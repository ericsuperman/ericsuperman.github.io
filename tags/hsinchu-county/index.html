餐廳名稱: 現流魚海鮮餐廳 地 址: 新竹縣新豐鄉新豐村14鄰346號 營業時間: 週四至週二 10:00-19:00 週三 公休 電話 : 0975 345 723 今天從平時收集的 疑似美食名單 挑一間海鮮餐廳來踩雷 - 位於新豐鄉紅毛港的 現流魚海鮮餐廳。今天吃完的感覺是，菜色還行，份量比一般大， 算起來 CP值挺好的，比風景區那些餐廳強多了。只是 環境是完全沒裝潢，吃飯氣氛 跟 當時的客人很有關，客人吵雜，氣氛就差，客人安靜吃飯，氣氛就還可以。 鐵皮屋、水泥地，毫無裝潢可言，桌椅也很普通，不過我比較重視好吃不好吃，這樣的環境還能接受，老闆還宣稱夏天有冷氣。下圖白頭髮的就是老闆。 今天用餐的前半段，這桌客人似乎是在地的親友們聚餐，大人高興地大嗓門聊天，小朋友放任他們跑來跑去嬉鬧，氣氛超級差的。 更糟的是，有人抽菸，可能是後門出去的樹蔭下，不知道是哪桌的，後院也有露天座位。可是後門沒關啊，走出後門就抽菸，煙味還不就飄進來，煩死了。 所幸用餐後半段，客人都吃飽離去，只剩後院還有一些客人，也沒煙味了，總算能悠閒地享用海鮮了。之後有再來幾組客人，大都是三、四人，自然是不會像十幾個人這樣大聲喧嘩。 鄉下海鮮店，菜單就牆壁上這種，但是也只是參考用，直接看魚貨點，最準確。 第二面菜單牆，仍然是參考用。像 炒海瓜子 已經賣完了。所有海鮮食材，都在下面兩個大冰櫃。老闆似乎有些門路，這些魚貨都是海釣的。餐廳名字就是標榜 現流魚。 店家餐具、白飯是自助式的，在後門的附近。白飯不算錢，免費供應。第一次來訪不知道，老闆、服務生又通常很忙，開始時客人多，不好意思到處看，吃到一半才發現。 炸蚵酥。很標準，跟一般餐廳差不多。 炸小卷。外面比較少見，一樣是配胡椒鹽，還不錯。跟炸蚵酥有點類似，可是沒辦法，之前做功課時，有人說這道好吃。 小朋友愛吃生魚片，老闆就推薦紅甘 一魚兩吃 - 切生魚片，剩下煮魚湯。 紅甘生魚片感覺有用熱水燙魚皮的部分，是一般餐廳常見的 湯霜法 手法，這是加速魚肉的熟成，讓生魚片更好吃。 這盤生魚片，撒了大量碎冰鎮住，還蠻鮮甜的。 紅甘魚湯 好的是感覺不到魚腥味，不過酒味就稍微濃一點，店家大概是用酒去腥味吧。只是魚湯的魚肉，都是切生魚片剩的，有一些魚骨頭、魚刺，吃起來有點累。 老闆炫耀說今天進了五尾紅甘，已經賣出四尾了。 炒麵，網路上有人推薦，不過吃起來還好，很正常。 老闆、老闆娘的理念就是不重裝潢，將實在的好吃海鮮，呈現給客人。現流魚海鮮餐廳差不多就是這樣子，我是認同也接受這樣的理念。 不過我覺得好吃度 沒有到 基隆安一五郎 的水準，那就不大值得忍受這樣的環境，特別是還有人抽菸的話，那就完全不能忍。 如果各位到紅毛港玩，其實是可以順便來用餐，專程前來就大可不必了。 Read more →

[竹北] HEA Bistro 小放空 • 手工料理 • 微醺

Sun Oct 13, 2019 by Super Cat in  Gourmet  Hsinchu county, Zhubei, Italian Food

餐廳名稱: HEA Bistro 小放空 • 手工料理 • 微醺 地 址: 新竹縣竹北市自強五路192號 營業時間: 週三至週一 11:30-14:00 17:30-22:00 週二 公休 電話 : 03 668 1191 女王大人在附近溜小孩時，發現這間試營運的餐廳。從外觀看，挺氣派的。回家上網查，有FB官方粉絲頁，其他資料很少， 不過心中有點譜了，今天準備好錢，要來當幾小時的有錢人。 大門口照片，門前有個風水學喜歡的明堂，有點開闊的氣勢。 對面是紐約新和食，如果附近找不到停車的地方，也許要轉到 成功11街 皇膳湯包 對面，有個私人停車場。 私人停車場 再過去有個公有停車場，但因為還沒開始收費，所以常常停滿，去全聯的客人也搶著停這個公有停車場。 進門第一區，牆壁上是大大的 HEA BISTRO 金字招牌。 今天進門時，剛好有位外國朋友，帶著小孩來外帶。來之前已經從網路知道老闆娘是香港人，老闆娘以流利英文跟外國朋友交談、接訂單，看起來老闆娘自己也是周遊列國過的，英文極佳。 下面照片是老闆娘領軍幾位娘子軍，認真地製作餐點。 大部分的座位都像是吧檯位。 除了進門附近這張桌子，勉強算是個六人桌。 這裡是高級餐廳，每人座位都有一整組餐具、餐巾。可能一組客人有一壺水吧，我拿過來跟餐具一起拍照。 老闆娘還在試營運改菜單當中，像多利魚似乎就是今天才放進去的，午餐目前只供應 Tapas，沙拉，義大利麵。 不過還好我點了，後面再解釋。 這些菜單，就是參考而已，應該還會常常更動，FB官方粉絲頁目前也還沒放上完整菜單。 老闆娘介紹菜單到飲品類的時候，索性把茶車推過來介紹。好幾款 英國Harrods百貨公司 出品的茶，然後好幾款台灣知名的茶廠、茶店的茶，例如 日月老茶廠。 老闆娘興奮地說 澄月紅玉 是台北一間有名的茶店嚴選的，在誠品有設櫃，店名講很快，我記不起來。 說她媽媽來拜訪她兩天，第一天是泡 日月老茶廠的紅玉，第二天她泡 澄月紅玉，她媽媽跟她說昨天好喝，可是今天更好喝。講的我都想點一壺來喝了。 “新鮮無花果，蜂蜜核桃，貝比嫩葉” 沙拉。新鮮無花果說是國內農場找來的，有機無毒的栽培方式。香港人老闆娘喜歡使用在地食材，真是棒。 今天點很多種 Tapas ，這道是 “西班牙火腿，新鮮無花果 普切塔”，普切塔其實就是義大利的麵包抹大蒜加番茄，當然之後演變成番茄換成很多別種食物。 那個西班牙火腿鹹香好吃，意猶未盡。 兩道 Tapas 裝在同個盤子， “香草紐西蘭小羔羊排棒棒” 與 “南乳烤雞翅”， 小羔羊排棒棒 普通，雖然很好的是沒羊騷味，可是肉沒有醃過，光靠表面味道還是不夠。 南乳烤雞翅 也是很正常，高級餐廳應該要再更有特色一點。 在吃完上面兩道Tapas後，老闆娘拿出乾紙巾，當時高度可能不到1公分。淋上玫瑰水後，長成如下圖的約 6、7 公分長。這是用手拿羊排、雞翅吃，專門搭配的紙巾。 使用起來到處留下濃郁的玫瑰香味，小朋友開玩笑想拿來敷臉。我在其他高級餐廳，也很少看到這種貼心的服務。 “烤白蝦，泰式辣椒青檸汁”，烤到蝦頭也能吃，那個醬汁調得很好，清爽、辣、酸 都有。粗鹽感覺是裝飾而已，沒什麼用。 小朋友堅持點的松露薯條。台北 知名高級餐廳 Tutto Bello ，也有松露薯條，比這邊貴、大份，可是也好吃很多，不大有必要吃這裡的。不過沒辦法，小朋友堅持吃這個，她才能專心念書。 唉，當然吃完還是沒有認真念書。 “蘑菇，奶油，帕瑪森起司，尖管麵” ，我覺得不夠鹹，醬汁太少，但她的派別似乎是義大利麵收得很乾，再多一點鹹一點的醬汁，覺得會好一點。 今天點的兩道義大利麵，感覺都是清淡派，很不適合我這種重口味的。 “蒜頭，辣椒，橄欖油 義大利麵” ，老闆娘事先來詢問辣度，蠻貼心的，不知道如何講，就說照烤白蝦的辣度，不過還是感覺還是偏辣一點，不過這點沒關係。 這道乾炒的麵，有達到清爽的感覺，很好。可是問題仍然是不夠鹹，難道女主廚先天就是不肯放鹽嗎，看來下次點義大利麵要叮囑一下了。 今日第一名 - “檸檬，法國奶油香煎多利魚”，還好有點。原來中午時段 店家目前只供應 Tapas、Pasta、沙拉。老闆娘問願不願意等，今天沒事，當然願意，老闆娘就去製作了。 上菜時間也還好啊，不覺得等很久，可能因為只剩我們一組客人。 Read more →
