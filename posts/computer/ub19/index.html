Ubuntu 19.04 開箱

Sat May 04, 2019 by Eric Chang in  Computer  Linux

Ubuntu 19.04 正式推出了，這一版不是 Long Term Support 版， 不過還是用 VirtualBox 玩玩看 Ubuntu Server好了， 看看那些東西進步了。

先新增一台虛擬PC，我的Host PC裝了很多ram， 所以這裡就很大方地給這台虛擬PC 4GB ram。

硬碟則是配了 128GB，反正VirtualBox對虛擬硬碟是 實際用多少才配置多少，一開始不會太大。如果想要玩 編譯 Linux Kernel 或 OpenWRT 的話，預設的10GB是不夠的。

再來，反正是在家裡隨便玩玩，我習慣模擬兩張網卡， 都用Bridge方式接到內網，

Ubuntu 安裝後，一張網卡配 191.168.1.x 的IP， 另一張網卡配 10.10.10.x 的IP， 如果要測試 NAT 功能，或是其他 gateway的功能， 兩張網卡才能做。

最後，虛擬光碟機要接到 ubuntu-19.04-live-server-amd64.iso 檔案，就能開機了。

第一次開機，從虛擬光碟boot，進入安裝程式， 安裝步驟雖然有14步，大部分預設值都沒問題， 通常都直接按Enter進到下一步。

第0步，選英文，直接按下去。

第1步，安裝程式 在 GitHub 有新版，問要不要換。 這個安裝光碟在開機時就利用DHCP連上網路了， 所以可以去GitHub看看有無新版， 網路都連好了，安裝程式也不大，當然就換新的安裝程式啊! Enter按下去。

第2步，是否確認是英文鍵盤， Enter按下去。

第3步，Enter按下去。

第4步，兩張網卡預設為用 DHCP 。這個我要改 ……

我玩這份 Ubuntu 就是要當 Server ，必須是固定IP。

選 Manual 進入 static IP 設定頁。

如下圖，把每個欄位都填一填。

第二張網卡也如法炮製，但是就不要跟第一張網卡同網段了， 未來利用這兩張網卡，兩個網段，可以來玩一些閘道器的功能。

第5步，Enter直接按下去，除非你的網路需要 proxy 才能連到 Internet。

第6步，Enter直接按下去，除非你知道有更快的 Ubuntu Mirror 站台。

我目前的設定，讓安裝程式跳過第7步了，我也不知道那是啥。

第8步，因為我是玩一玩而已，”Use An Entire Disk” 就夠了。 如果是 Production 環境，應該是要規劃個 LVM 。

選即將要格式化的硬碟，雖然說這個步驟要很小心， 可是這台虛擬PC只看的到這台虛擬硬碟啊， 沒別台了。 如果能看到 Host PC 的硬碟，那才奇怪。

在 “Done” 按下Enter，就照上面顯示的Partition分法去格式化了， 這個步驟仍然要很小心。

怕出錯，再確認一次，因為一旦格式化，資料是救不回來的。

沒問題的話，選Continue，按下Enter。

因為格式化之後，直接就開始安裝了，所以還有幾個問題要先問，

第一個問題，就是決定這個系統的第一個 user。 因為Ubuntu 預設不能直接使用root，很多事都需要這個 user 用 sudo 去做。

第二個問題，裝Secure Shell Server，當然要，這樣我才能在 Host PC 用 ssh client 連進Server 設定東西，都用Console 不方便。

第三個問題， Ubuntu 整理了很多 Server 常用的套件組，用這個安裝 會非常省事。不過這個未來可以再選，這次就什麼都不選。

以上問題都答了，就開始格式化，開始安裝全部的檔案。過程需要好幾分鐘。

裝好了，就出現下面畫面，等你按Reboot。

一份安裝好的 Ubuntu 19.04 Server，開機成功，可以開始用了。

新的Linux Distribution大部分開機都很快，其實這是高手們分析了開機過程， 某些工作放在等候Login訊息之後做也沒關係， 所以看到 login: 後，一些boot訊息還是慢慢跑出來， 但無所謂，就登入，開始做事吧。

裝好 tasksel 後，執行它， LAMP Server 打上星號，按OK，幾分鐘就裝好了。

如果是 tasksel 沒包括的，那當然就是用 apt 一個一個裝了。 例如: gcc, g++, gnu make

前一篇技術文章提到的 VirtualBox Shared Folder ，需要插入 VirtualBox 開發的一個 Linux Kernel Module， 這個模組的安裝程式與原始碼，都放在VirtualBox自備的 Guest Additions 虛擬光碟。

但安裝程式需要gcc 等工具，去編譯這個模組， 所以Ubuntu Server 還是得安裝這些基本的開發套件。 

root@www:~# apt install gcc g++ make

還有一個 snap 格式，有些package的最新版是用snap格式包的， 例如: golang 。 Server類型的程式用 golang 開發有許多好處。 Ubuntu自己的 Golang package 不會瘋狂追逐新版，可是 Golang 還 在積極開發中，不用新版的話，有點沒意思。

可以看到下圖用snap裝的話，版本為 1.12.2 ，在此時是蠻新的。 

root@www:~# snap install --classic go
go 1.12.2 from Michael Hudson-Doyle (mwhudson) installed
root@www:~#

以上，先裝到這裡。
